# Konsep compiler di c++

file cpp -(preprocessing)-> file cpp + include (cth:iostream) -(compiling)-> file obj -(linking)-> file exe (obj+obj+....)

# Step by Step

1. Buat file .cpp
2. Setelah file diisi, Tekan run dan pilih g++ build 
3. Setelah muncul folder .vscode, tulis di terminal "g++ (nama_file)"
4. Setelah muncul file .exe, jalankan program di terminal dengan ".\(nama_file exe)

*langkah 2-3 bisa dipangkas menggunakan ctrl+shift+b
